## [1.0.0] - 2026-02-09

### Added
- **Theorycrafting Tool**: Complete build calculator for Aion 2
- **Skill Management**: Manage active, passive, and stigma skills
- **Build System**: Create and customize character builds
- **Sharing System**: Share builds with community via unique URLs
- **Community Features**: Browse and discover community builds
- **Skill Database**: Comprehensive skill tree with requirements and effects
- **User Interface**: Intuitive UI for build creation and management

### Security
- User authentication for build saving
- Build ownership and edit controls
