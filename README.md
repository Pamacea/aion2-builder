# BAHION - AION 2 BUILDER

**BAHION** is a builder that focuses on *theorycrafting* skills: active, passive, and stigmas. It allows you to *think about* and *share your ideas* and solutions to in-game issues.


## Features

- A short description of each aion class
- An aesthetic and quality of life builder
- Live caracteristics preview
- Autosave builder
- Mobile friendly


## Website

## Tech Stack

**Client:** React, Nextjs, TailwindCSS

**Server:** Node, Express, PostgreSQL, Prisma
